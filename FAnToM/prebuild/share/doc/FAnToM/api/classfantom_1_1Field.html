Detailed Description

template<size_t D, class T>
class fantom::Field< D, T >

An analytical or interpolated tensor field.

A tensor field is a function G -> V from a domain (kernel) G to a co-domain (range, value) V, where V is usually a set of tensors of same order and dimension. Note: T must support addition (T+T) and scalar multiplication (double*T) for the evaluator to work.
