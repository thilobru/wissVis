Detailed Description

template<class T>
class fantom::ValueArray< T >

an array which stores data

The purpose of this class is an interface to an array of values that abstracts from the precision in which values are actually stored.

Template Parameters
Tthe type of data.
Exceptions
fantom::out_of_rangewhen accessing invalid indices (only in debug build)
Note
All members are thread-safe.
