Detailed Description

Basic class for implementing Shells.

The shell contains a history area that can be used to display previous commands and output of executing them, as well as a command prompt area, where a new command can be entered interactively.

The Shell takes care of all user interaction needed to manipulate the content of the command prompt. All keyboard events are also relayed to child classes via the keyPress/keyRelease functions. Therefore, these functions can be used to implement everything related to the execution of commands.
