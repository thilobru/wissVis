Detailed Description

template<class InputType>
class fantom::PythonConverterRegister< InputType >

registers a python converter for an Input

Static variables of this class are used to automatically register python converters. Example:

PythonConverterRegister<MyInputType> dummy();

After registration, the converter type will be used to interpret python scripts.

Template Parameters
InputTypethe type of Input to register. It is required to be a valid subtype of InputBase.
