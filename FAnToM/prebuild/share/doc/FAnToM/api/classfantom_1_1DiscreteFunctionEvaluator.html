Detailed Description

template<typename T>
class fantom::DiscreteFunctionEvaluator< T >

discrete evaluator

The purpose of an evaluator is to divide a value array into time-slices and allowing an access to the values by value and time index.

Exceptions
fantom::out_of_rangewhen accessing invalid indices (only in debug build)
Note
Use one evaluator per thread.
