Detailed Description

template<class T, std::size_t... N>
class fantom::Tensor< T, N >

tensor

Objects of this class are tensors, i.e. generalizations of mathematical scalars, vectors, matrices. The class uses variadic templates to encode both order and dimension of scalars, e.g.

  • Tensor<float> is a single precision scalar type,
  • Tensor<double, 3> is a double precision 3-D vector type,
  • Tensor<complex<float>, 2, 3> is a single precision complex 2 by 3 matrix type. The matrix has 2 rows and 3 columns.
  • Tensor<int, 2, 3, 4> is a 2 by 3 by 4 tensor (3rd order) on integers.
  • ...

The memory layout of coefficients matches the layout of C++ multidimensional array, e.g. Tensor<int, 2, 3, 4> is laid out exactly as int[2][3][4]. Accessing coefficients can be done with operator()() which takes a list of indices to address coefficients. An operator[]() is provided as well but works only with vectors.
