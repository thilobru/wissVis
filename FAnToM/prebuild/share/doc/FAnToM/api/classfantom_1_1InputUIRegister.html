Detailed Description

template<class InputUIType>
class fantom::InputUIRegister< InputUIType >

registers an InputUI type at the kernel

Static variables of this class are used to automatically register InputUIs. Example:

InputUIRegister<MyInputUI> dummy();

After registration, the InputUI type can be used by the GUI to handle algorithm inputs.

Template Parameters
InputUITypethe type of InputUI to register. It is required to be a valid subtype of InputUI as documented at the InputUI class.
