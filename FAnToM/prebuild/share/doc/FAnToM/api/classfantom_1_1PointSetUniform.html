Detailed Description

template<size_t D>
class fantom::PointSetUniform< D >

A uniform point set. Indices are mapped to positions according to the documentation in DiscreteDomain. The positions in the ValueArray are ordered by structuring dimensions. The first dimension is incremented first, then the second, then the third and so on.
