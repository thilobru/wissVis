 Plugin DevelopmentFAnToM can be extended by writing plugins. A FAnToM plugin is a shared object/dynamically linked library that contains one or more algorithms
 Mathematical Data StructuresFAntoM provides its own set of mathematical data structures. The core data structure is a tensor, which is a generalization of scalars, vectors, and matrices. A tensor may be defined on any body of numbers including complex values via its first template parameter. A tensor has an order (0 - scalar, 1 - vector, 2 - matrix) and dimensions, which are specified using variadic templates. The order of the tensor is automatically inferred from the number of template parameters. FAnToM does not (yet) support efficient storage for symmetric or diagonal tensors
