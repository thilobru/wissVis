Detailed Description

template<typename T>
class fantom::FunctionEvaluator< T >

discrete evaluator with time interpolation

The purpose of an evaluator is to divide a value array into time-slices and allowing an access to the values which can perform interpolation across time slices.

Exceptions
fantom::out_of_rangewhen accessing invalid indices (only in debug build)
Note
Use one evaluator per thread.
