◆ init()

virtual void fantom::GlPainter::init ( const std::function< void() > &  update)
pure virtual

inititalize OpenGL painter

This member is used to initialize the OpenGL context, e.g. textures and display lists. It will be executed exactly once and also informs about the callback that can be used to request paint().

Parameters
updatethis function may be called by the painter (or some other party) to request a call to paint(). The request is scheduled and executed later.
Precondition
OpenGL content is current, init(), resize() and paint() have not been called before.
Postcondition
nothing.
Note
Thread-safe.
Exceptions
nothing.

virtual void fantom::GlPainter::paint ( )
pure virtual

Repaint drawing area using OpenGL.

This member is called if the window system repaints windows or if it was requested via update() (see init()).

Precondition
OpenGL content is current, init() has been called once before, paint() and resize() are not executed currently in other threads.

nothing.
