Cells is an index-accessible array of size size of cells. A cell must provide the functions

  • c.index( size_t i ), which returns the index of the i-th vertex in the cell
  • c.numVertices(), which returns the number of vertices in the cell

Points is an index-accessible array of point locations. A point must provide an index operator p( dim ), which returns the dim-th coordinate of the point for 0 <= dim < D.
