Detailed Description

Implements the CellTree search structure to locate cells.

This is an implementation of the data structure and data construction algorithm for the fast cell location as presented in "Fast, Memory-Efficient Cell Location in Unstructured Grids for Visualization" by Christoph Garth and Kenneth I. Joy presented at VisWeek 2011
